## 🚀 Deployment to IPFS

Untuk menjalankan Explorer ini dalam rangkaian desentralisasi:

1. Add folder ke IPFS Node:
```bash
ipfs add -r Sovereign-Explorer/

```


2. Akses melalui gateway pilihan anda:
```text
[https://ipfs.io/ipfs/](https://ipfs.io/ipfs/)<YOUR_CID>

```
